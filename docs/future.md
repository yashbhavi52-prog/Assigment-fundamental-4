# Future Improvements

This document outlines possible enhancements and extensions for the calculator project.

## Planned Features
- Add advanced operations such as:
  - Square root
  - Power (exponentiation)
  - Modulus (remainder)
- Support for multiple calculations in a single run.
- Option to save calculation history.

## User Interface
- Build a graphical user interface (GUI) using Tkinter.
- Create a web-based version using HTML/CSS/JavaScript.

## Testing & Reliability
- Add unit tests to ensure correctness of all operations.
- Improve error handling and input validation.

## Collaboration
- Explore branching workflows for new features.
- Use pull requests and code reviews for team collaboration.

## Long-Term Goals
- Package the calculator as a Python module.
- Publish the project for others to install and use.
